# Code of Conduct

CoStaff adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct.

In short: be respectful, be constructive, focus on what's best for the
community. The full text — including expected behavior, unacceptable
behavior, and enforcement guidelines — lives at the link above.

## Reporting

Reports go to the maintainer at the contact email listed in the
project's repository (or as `SECURITY.md` for sensitive reports).
Reports will be reviewed promptly and confidentially.

## Scope

This Code applies in all project spaces — repository issues, pull
requests, discussions, Discord, and any public representation of the
project.
